# Project Eventos Sociais
Uma aplicação full stack para gestão de eventos sociais. Pessoa cliente se cadastra no site e solicita orçamento, selecionando o evento desejado e outros serviços e produtos opcionais. A pessoa administradora fecha o orçamento e posteriormente a pessoa cliente aprova (ou não).
